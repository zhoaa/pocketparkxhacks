# XHacks 2021 project

This app is run on the MERN (MongoDB, Express, React, Node) stack. The server lives in the `server` folder and the client lives in the `client` folder.

### Getting started

In both the `server` and `client` folders, in separate terminals:

```
$ npm install
$ npm run start
```
